Bem-vindo ao PEDAGOGO.AI

Este assistente vai configurar todo o sistema de automação pedagógica no seu Google Workspace.

O setup vai executar:

🔍 Verificar pré-requisitos (chave Gemini)
📁 Criar estrutura de pastas no Drive
📊 Criar planilhas-mestre (BNCC, Questões, Turmas, Resultados)
🔒 Aplicar proteções LGPD
⚡ Instalar triggers automáticos
📝 Criar formulários do sistema

💡 A chave Gemini pode ser configurada antes ou após o setup.
Acesse: painel → 🔧 Configurar → Chave API Gemini
